# MANG369 TikTok Growth Strategy 🚀

Este repositorio contiene el código fuente de la página web estática de la estrategia de crecimiento en TikTok para **MANG369**, optimizada para ser publicada de forma gratuita y directa en **GitHub Pages**.

## 📁 Estructura del Proyecto
- `index.html`: Página principal con diseño moderno en modo oscuro, cyberpunk/esotérico, adaptada para móviles y escritorio.
- `assets/logo.png`: Logotipo oficial de MANG369.
- `assets/qr.jpg`: Código QR de invitación al servidor de Discord.

## 🚀 Publicado en GitHub Pages:
1. Archivos en un repositorio en GitHub ( `mang369-tiktok-strategy`).
2.**Build and deployment**, rama `main`. y la carpeta `/ (root)`.
3. **Pages** web online.
